# True Random Number Generator - TRNG Resources and References

There are 2 Tiny Tapeout projects developed in parallel:

- **TT-GF180**: [github.com/gojimmypi/ttgf-UART-FSM-TRNG-Lab](https://github.com/gojimmypi/ttgf-UART-FSM-TRNG-Lab), 
submitted in [Tiny Tapeout GF 26a](https://app.tinytapeout.com/shuttles/ttgf26a), 
see [Project #4337](https://app.tinytapeout.com/projects/4337). 
Purchase one [here](https://app.tinytapeout.com/prepurchase?shuttle=ttgf26a&tiles=0&pcbs=1).

- **TT-SKY180**: [github.com/gojimmypi/ttsky-UART-FSM-TRNG-Lab](https://github.com/gojimmypi/ttsky-UART-FSM-TRNG-Lab), 
draft, not submitted in [ttsky26a](https://app.tinytapeout.com/shuttles/ttsky26a), [Project #4338](https://app.tinytapeout.com/projects/4338)
